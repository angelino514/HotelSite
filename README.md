Sistema de Hotel - Projeto de Demonstração

Autor: Angelino Eurico

Objetivo:
Este projeto foi desenvolvido para demonstração de competências em desenvolvimento web e para apresentação a potenciais clientes de freelancing.

Funcionalidades:
- Apresentação do hotel
- Simulação de reserva/agendamento
- Interface responsiva
- Navegação intuitiva

Finalidade:
Criar oportunidades profissionais, expandir networking e servir como portfólio para futuros projetos comerciais.
